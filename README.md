## 진행 방법 
- `Issue` 에 각 `mission` 을 등록하였습니다. 

- 작업을 수행할 때는 `branch` 를 만들어서 작업을 수행합니다. `feature/[your-name]` 으로 작성합니다. 

- 각 `mission` 수행 시 `pull request` 요청을 부탁드립니다. 

- 간단한 `code review` 를 통해 `흐름`을 명확히 파악해 봅시다.
